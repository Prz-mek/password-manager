# password-manager
It's **not** completely safe password manager.

## features
* hashing master password
* encrypting and decrypting passwords to external services
* master password strength validation
* sign in delay


(in progress)

## running
To run this code Docker have to be installed:

```
docker-compose-up
```

or use flask server.
